# tf-handson
初心者向けterraformハンズオン用リポジトリ
[Terraformハンズオン](https://mmmcorp.esa.io/posts/4476)

# 環境構築

- Docker for Macをインストールする。
- リポジトリをクローンする

```bash
git clone git@github.com:mmmcorp/tf-handson.git
cd tf-handson
```

- `env.sample` ファイルをコピーして、 `.env` を作成し、そこにAWSアクセスキーなど入力する

```bash
cp env.sample .env
```
